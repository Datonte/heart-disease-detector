# HeartFelt Hospital Management System

A Flask-based Hospital Management System with integrated Heart Disease Prediction using Machine Learning.

## Features
- **User Authentication**: Secure login and registration for staff.
- **Patient Management**: Add, view, and update patient records.
- **ML Integration**: Predict heart disease risk using a pre-trained model.
- **Appointments**: Book and manage appointments.
- **Dashboard**: Real-time statistics and charts.
- **Reports**: Generate simple reports and view high-risk cases.

## Setup & Run

1. **Install Dependencies**:
   ```bash
   pip install -r app/requirements.txt
   ```

2. **Run the Application**:
   ```bash
   python run.py
   ```

3. **Access the App**:
   Open your browser and go to `http://127.0.0.1:8080`.

## Default Admin/Staff
Register a new account on the login page to get started.

## Model
The system expects `heart_model.pkl` in the root directory.
